JS Objects #1

Что такое объект в JS ?

Ассоциативный массив – структура данных, в которой можно хранить любые данные в формате ключ-значение.

https://learn.javascript.ru/object

Создание объекта в JS

				var foo = {};
				console.log(foo); // => {}
			
				var foo = {
					prop: 'some string'
				};
				console.log(foo); // => {prop: 'some string'}

Глобальные переменные как свойства объекта window

				document // глобальная переменная
				window // другая глобальная переменная =)

Все глобальные переменные являются свойствами объекта window 

				console.log(foo) // => ReferenceError: foo is not defined
				console.log(window.foo) // => undefined
				var foo = 123;
				console.log(foo) // => 123
				console.log(window.foo) // => 123

Задание свойств объекта в JS

				var foo = {};
				console.log(foo); // => {}
				console.log(foo.prop); // => undefined
				

				foo.prop = 'some value'
				console.log(foo); // => {prop: 'some value'}
				console.log(foo.prop); // => 'some value'

Обраный случай с определением переменной - очень плохой код =)

			console.log(foo) // => ReferenceError: foo is not defined
			console.log(window.foo) // => undefined
			window.foo = 123;
			console.log(foo) // => 123
			console.log(window.foo) // => 123

Допустимые значения для свойств объектов

В качестве значения свойства может выступать любое допустимое в JS значение

				var foo = {           
					subObj: {           
						text: 'string',   
						num: 123,         
						subObj: {         
							isTrue: false   
						}                 
					},                  
					ids: [1, 2, 3, 4],  
					logIt: function(){  
						console.log(this) 
					}                   
				};

Сравнение объектов

				var foo = {num: 123};
				var too = {num: 123};
				console.log(foo === too); // => false
				console.log(foo == too); // => false

При создании объекта в переменную записывается ссылка на объект в памяти, а не сам объект.

Единственный вариант, когда при простой проверке на равество, объекты равны, это когда они имеют одну и ту же ссылку.

				var foo = {num: 123};
				var too = foo;
				console.log(foo === too); // => true

Изменение примитивов vs Изменение объектов

					var num1 = 123;
					var num2 = num1;
					console.log(num1); // 123
					console.log(num2); // 123
					var num2 = 234
					console.log(num1); // => 123
					console.log(num2); // => 234
				
					var foo = {num: 123}
					var too = foo
					console.log(foo); // => {num: 123}
					console.log(too); // => {num: 123}
					too.num = 234
					console.log(too); // => {num: 234}
					console.log(foo); // => {num: 234}
