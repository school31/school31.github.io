AJAX - что такое?

AJAX (аббревиатура от «Asynchronous Javascript And Xml») – технология обращения к серверу без перезагрузки страницы.

За счет этого уменьшается время отклика и веб-приложение по интерактивности больше напоминает десктоп.

Несмотря на то, что в названии технологии присутствует буква X (от слова XML), использовать XML вовсе не обязательно. Под AJAX подразумевают любое общение с сервером без перезагрузки страницы, организованное при помощи JavaScript.

https://learn.javascript.ru/ajax

Варианты написания запроса

  • Нативный XMLHttpRequest - https://learn.javascript.ru/ajax-xmlhttprequest
  • jQuery
  • axios
  • ...куча других библиотек
  • fetch - https://learn.javascript.ru/fetch - нативный способ, но на данный момент слабо поддерживается браузерами

AJAX на примере jQuery

http://api.jquery.com/jquery.ajax/

			$.ajax({ // обращаемся к методу ajax
				url: 'site.com/api/v1/home',
				// url где "лежит" то, что нам нужно
				method: method,  // get, post, patch, put, ...
				success: function(data){ /* обрабатываем ответ */ },
				error: function(xhr){ /* обрабатываем ошибку */ }
			})
