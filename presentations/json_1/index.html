JavaScript Object Notation

JSON - что это?

JSON - текстовый формат обмена данными, основанный на JavaScript. Чаще всего, когда вы пишите ajax запрос, вы ожидаете получить данные именно в формате JSON.

https://learn.javascript.ru/json

JSON - особенности

JSON напоминает обычный javascript-объект, но обладает рядом особенностей.

В качестве значений, в JSON, могут быть использованы только следующие значения:

  • Объект - {"ключ": значение}
  • Массив - [значение1, значение2, значение3, ...]
  • Число - 123
  • Литералы - true, false, null
  • Строки - "какая-то строка"

JSON - особенности #2

Все ключи и все значения, которые являются строками, в JSON должны быть обернуты в двойные кавычки - "foo"
