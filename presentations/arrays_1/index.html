JS Arrays #1

Что такое массив в JS ?

Массив – разновидность объекта, которая предназначена для хранения пронумерованных значений и предлагает дополнительные методы для удобного манипулирования такой коллекцией.

Они обычно используются для хранения упорядоченных коллекций данных, например – списка товаров на странице, студентов в группе и т.п.

https://learn.javascript.ru/array

Создание массива в JS

				var foo = [];
				console.log(foo); // => []
			
				var foo = [1, 2, 3]
				console.log(foo); // => [1, 2, 3]

Обращение к элементу массива

			var foo = [34, 45, 21, 76, 84];
			foo[0] // => 34
			foo[3] // => 76

Нумерация элементов массива начинается с нуля.

Длина массива и обращение к последнему элементу массива

			var foo = [34, 45, 21, 76, 84];
			foo.length // => 5
			foo[foo.length - 1] // => 84

Сравнение массивов

Массив это частный случай объекта, на него действуют те же правила сравнения, что и для всех объектов. То есть равны только те массивы, которые имеют одинаковую ссылку.

Изменение массива

			var foo = [1, 2, 3]
			foo[1] = 564;
			console.log(foo); // => [1, 564, 3]
			foo[foo.length] = 888;
			console.log(foo); // => [1, 564, 3, 888]
			foo[10] = 999;
			console.log(foo); // => [ 1, 2, 3, 888, <6 empty slots>, 999]

Изменение массива - #2

			var foo = [1, 2, 3];
			foo.push(10);
			foo; // => [1, 2, 3, 10]

Задание

Имеется определенный набор данных. Это объект. В нем есть вложенный объект с описанием новости и массив с комментариями. В описании новости есть: название, текст, дата публикации, картинка и объект с данными автора новости. В объекте автора новости есть имя и аватар. Каждый элемент массива комментариев это объект. У комментария есть текст и дата. Так же у комментария можеть быть автор. Автор так же является объектом с именем. Так же у автора может быть аватар.

  • lvl 1: написать функцию, которая будет генерировать html с новостью (заголовок, картинка, текст, дата публикации)
  • lvl 1: + функция должна генерировать список комментариев (текст, дата публикации)
  • lvl 2: + функция должна генерировать автора комментария если он есть и показывать заглушку, если автора нет (имя)
  • lvl 3: + функция должна показывать генерировать изображение с аватаром автора комментария, если он есть или показывать заглушку (аватар комментатора)
