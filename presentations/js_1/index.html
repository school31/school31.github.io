Типы данных в JS

  • Простые (примитивные) типы данных
  • Объекты

Простые (примитивные) типы

  • String - строка -> 'some string' ('string' или "string")
  • Number - число -> 123 (123 - число, "123" - строка)
  • Boolean - логическое значение -> true/false
  • null
  • undefined

Объекты

  • Object - Объекты -> {...}
  • Function - Функции -> function name(){...}
  • Array - Массивы -> [1, 2, 3, ...]
  • RegExp - Регулярные выражения -> /.../

Оператор typeof

				typeof 123       // => number
				typeof "123"     // => string
				typeof true      // => boolean
				typeof false     // => boolean
				typeof null      // => object
				typeof undefined // => undefined
			
				typeof Object    // => object
				typeof Function  // => object
				typeof Array     // => object
				typeof RegExp    // => object

Объявленние переменной в JS

Для объявления переменной используется ключевое слово "var" (от "variable" - "переменная") 

			var k;

Инициализация переменной в JS

Для инициализации (задания значения) используется оператор "="

			k = 10;

Динамическая типизация

			var k = 10;
			typeof k; // => number
			k = 'some string';
			typeof k; // => string

Объявление переменной в Java

			int x = 100
			float y = 123.45

Сложение в JS

Сложение чисел

			var n1 = 1;
			var n2 = 2;
			n1 + n2; // => 1 + 2 = 3
			n2 + n1; // => 2 + 1 = 3

Сложение строк (конкатенация)

			var s1 = 'some';
			var s2 = 'string';
			s1 + s2; // => 'some' + 'string' = 'somestring'
			s2 + s1; // => 'string' + 'some' = 'stringsome'

Слабая типизация

				var num = 123;
				var str = 'some string';
				num + str; // => 123 + 'some string' = '123some string'
				str + num; // => 'some string' + 123 = 'some string123'
				str * num; // => 'some string' * 123 = NaN

NaN - Not a Number

				var k = 'some string' * 123; // => NaN
				typeof k; // => number
