Что такое RegExp?

Регулярные выражения – мощное средство поиска и замены в строке. В JavaScript регулярные выражения реализованы отдельным объектом RegExp и интегрированы в методы строк.

https://learn.javascript.ru/regular-expressions-javascript

Создание RegExp в JS

			var foo = /abc 123/gim;
			typeof foo // => 'object'

Слеши '/' говорят JavaScript о том, что это регулярное выражение. Они играют здесь ту же роль, что и кавычки для обозначения строк или фигурные скобки для обозначения объектов.

Флаги RegExp в JS

			var foo = /.*/gim;

g - Если этот флаг есть, то регэксп ищет все совпадения, иначе – только первое.

i - Если этот флаг есть, то регэксп ищет независимо от регистра, то есть не различает между А и а.

m - Многострочный режим.

Классы RegExp в JS

/abc 123/

\d - любая цифра от 0 до 9 => /abc \d\d\d/

\w - любая латинская буква => /\w\w\w 123/

\s - любой пробельный символ, табуляция или перевод строки => /abc\s123/

\D, \W, \S - обратные проверки

. - любой символ кроме перевода строки

http://javascript.ru/regexp

Спецсимволы RegExp в JS

[ \ ^ $ . | ? * + ( )

/abc 123/

  • [abc] - любой из преведенных символов => /[abc][abc][abc] 123/
  • [^abc] - любой, кроме преведенных символов => /[^bc][^ac][^ab] 123/
  • \ - спецсимвол для экранирования спецсимвола.
    Например /./ - обозначает поиск любого символа.
    Например /\./ - обозначает поиск точки.
  • ^ - обозначение начала строки => /^abc/

Спецсимволы RegExp в JS - #2

[ \ ^ $ . | ? * + ( )

  • $ - обозначение конца строки => /abc$/
  • | - или. Например /red|green/
  • ? - обозначение необязательности элемента => /abc?/
  • * - обозначает повторение 0 или более раз
  • + - обозначает повторение 1 или более раз
  • ( ) - "запоминающие скобки"

Кватификаторы RegExp в JS

+, *, ? и {n}

{n} - позволяет найти точно количество повторений элемента

  • {0,} - от 0 повторений. Эквивалентно * => /acb*/ или /acb{0,}/
  • {1,} - от 1 повторений. Эквивалентно + => /acb+/ или /acb{1,}/
  • {0,1} - от 0 до 1 повторений. Эквивалентно ? => /acb?/ или /acb{0,1}/
  • {5,} - от 5 повторений, до беконечности => /b{5,}/
  • {5} - точно 5 повторений => /acbbbbb/ или /acb{5}/
  • {2,4} - от 2 до 4 повторений => /acb{2,4}/

Практика - #1

			'some string here'
			/some/   // => 'some string here'
			/me st/  // => 'some string here'
			/g he/   // => 'some string hee'
			/ring/   // => 'some string here'
			/strong/ // => 'some string here'
			/ere/    // => 'some string here'

Практика - #2

			'some 123 string 234 here'
			/^.{5}/    // => 'some 123 string 234 here'
			/\d+\s.*$/ // => 'some 123 string 234 here'
			/\d.*\d/   // => 'some 123 string 234 here'
			/\w+.+\d/  // => 'some 123 string 234 here'
			/\.*\d+/   // => 'some 123 string 234 here'
			/\d{2}/g   // => 'some 123 string 234 here'

Методы RegExp и String

https://learn.javascript.ru/regexp-methods

  • regExp.test(string) - проверяет соответствует ли строка регулярному выражению
    Например /test/.test('string for testing') => true => 'string for testing'
  • string.match(regExp) с флагом g - ищет все совпадения и возвращяет массив совпадений
    'some 123 string 234 here'.match(/^\w{1}|\w{3} 1|\d{2}\sstr|re$/g)
    [ "s", "ome 1", "23 str", "re" ]
  • string.match(regExp) без флага g - действует немного по другому

Методы RegExp и String - #2

string.replace(reg|str, str|func) 

			var str = 'some 123 string 234 here';
			str.replace('s', '0')       // => '0ome 123 string 234 here'
			str.replace(/s/g, '0')      // => '0ome 123 0tring 234 here'
			str.replace(/[2ei]/g, '0')  // => 'som0 103 str0ng 034 h0r0'
			str.replace(/\w/g, '0')     // => '0000 123 000000 234 0000'
			str.replace(/[SRM]/g, '0')  // => 'some 123 string 234 here'
			str.replace(/[SRM]/ig, '0') // => '0o0e 123 0t0ing 234 he0e'
