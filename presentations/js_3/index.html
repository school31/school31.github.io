Взятие остатка %

Оператор взятия остатка % интересен тем, что, несмотря на обозначение, никакого отношения к процентам не имеет.

			alert( 5 % 2 ); // 1, остаток от деления 5 на 2
			alert( 8 % 3 ); // 2, остаток от деления 8 на 3
			alert( 6 % 3 ); // 0, остаток от деления 6 на 3

Инкремент/декремент: ++, --

Одной из наиболее частых операций в JavaScript, как и во многих других языках программирования, является увеличение или уменьшение переменной на единицу.

Инкремент ++ увеличивает на 1: 
			var i = 2;
			i++; // более короткая запись для i = i + 1.
			alert(i); // 3

Инкремент/декремент: ++, --

Декремент -- увеличивает на 1: 
			var i = 2;
			i--; // более короткая запись для i = i - 1.
			alert(i); // 1

Инкремент/декремент: ++, --

Разница между i++ и ++i

			var i = 1;
			var a = ++i; // (*)

			alert(a); // 2

В строке (*) вызов ++i увеличит переменную, а затем вернёт ее значение в a. Так что в a попадёт значение i после увеличения.

Инкремент/декремент: ++, --

Разница между i++ и ++i

			var i = 1;
			var a = i++; // (*)

			alert(a); // 1
Постфиксная форма i++ отличается от префиксной ++i тем, что возвращает старое значение, бывшее до увеличения.

Что выведет alert()

			var i = 1;
			alert( 2 * ++i ); // 4

Что выведет alert()

			var i = 1;
			alert( 2 * i++ ); //  2

Операторы сравнения

Из математики

  • Больше/меньше: a > b, a < b.
  • Больше/меньше или равно: a >= b, a <= b.
  • Равно a == b. Для сравнения используется два символа равенства '='. Один символ a = b означал бы присваивание.
  • «Не равно». В математике он пишется как ≠, в JavaScript – знак равенства с восклицательным знаком перед ним !=.

Как и другие операторы, сравнение возвращает значение. Это значение имеет логический (boolean) тип.

Существует всего два логических значения:

  • true – имеет смысл «да», «верно», «истина».
  • false – означает «нет», «неверно», «ложь».
			alert( 2 > 1 );  // true
			alert( 2 == 1 ); // false
			alert( 2 != 1 ); // true

Сравнение строк

Строки сравниваются побуквенно

			alert( 'Б' > 'А' ); // true
			alert( 'Вася' > 'Ваня' ); // true, т.к. 'с' > 'н'
			alert( 'Привет' > 'Прив' ); 
			// true, так как 'е' больше чем "ничего".

Важно!

Обычно мы получаем значения от посетителя в виде строк. Например, prompt возвращает строку, которую ввел посетитель. Числа, полученные таким образом, в виде строк сравнивать нельзя, результат будет неверен. Например:

			alert( "2" > "14" ); // true
			
Как сделать правильно?

			alert( +"2" > +"14" ); // false
