Функции - практика #1

Что не так с этой функцией? #1

			var k = 111;      
			function foo(a){  
				k = 222;        
				return a * a;   
			}

Функция имеет побочное действие.

Что не так с этой функцией? #2

			var k = 222;     
			function foo(a){ 
				return a * k;  
			}

Функция работает с данными, переменной k, которя не была явно передена в нее.

Что не так с этой функцией? #3

			function foo(a){               
				if (Math.random() * 2 > 1) { 
					return a + 1;              
				} else {                     
					return a - 1;              
				}                            
			}

При одном и том же аргументе, функция может возвращать разные значения.

Чему равна переменная k? #1

			var k = 111;                         
			foo();                               
			k // => 111
			function foo(){                       
				k === 222                           
			}

Функция foo ничего не меняет

Чему равна переменная k? #2

			var k = 111;                           
			k; // => 111 
			function foo(){                        
				k = 222;                             
			}

Функция foo может изменить k, но она не вызывается и ничего не меняется.

Чему равна переменная k? #3

			var k = 111;                                         
			k; // => 111               
			foo();                                               
			function foo(){                                      
				var k;                                              
				k = 222                                            
			}

Функция foo определяет собственную переменную k, поэтому переменная k из глобальной области не меняется.
